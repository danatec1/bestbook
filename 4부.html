제 4부. ActiveX 초급과정

제 4부 ActiveX 초급과정은 2부에서 학습한 텍스트박스, 커맨드버튼, 레이블 컨트롤과 더불어 비주얼베이직이 기본적으로 제공하는 컨트롤에 대해 학습합니다. 기본이 충실한 사람치고 실력없는 사람이 없습니다. 제 4부는 다음과 같이 다섯 개의 장으로 이루어져 있습니다.

1장. 기본 ActiveX 컨트롤 Ⅰ
2장. 기본 ActiveX Control Ⅱ
3장. 파일 시스템 컨트롤
4장. Common Dialog Box 컨트롤
5장. FlexGrid 컨트롤

ActiveX는 마이크로소프트가 발표한 모든 pc 관련 기술에 인터넷 환경을 접목시킨 것으로, 궁극적인 목적은 WWW(World Wide Web)의 문서들과 함께 사용할 수 있도록 만든 것입니다. ActiveX Control이라 하면 COM(Component Object Model)을 기반으로 한 인터페이스를 준수하며, OLE 컨트롤이나 Java 애플릿의 슈퍼 클래스 역할을 합니다. 매우 어렵게 느껴지나요? 어렵지 않습니다. 단순히 여러분이 비주얼베이직에서 사용하고 있는 컨트롤들이 ActiveX 컨트롤이라는 것을 알면 됩니다. 또한, 이 ActiveX 컨트롤은 인터넷 환경에서 사용할 수 있다는 것을 기억하기 바랍니다. 제공되는 간단한 예제들을 통해 비주얼베이직이 제공하는 기본적인 ActiveX 컨트롤을 익혀보도록 하겠습니다.
